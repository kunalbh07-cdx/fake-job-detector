# AI Based Fake Job & Internship Detection System

This project detects fake job and internship postings using Machine Learning and rule-based red flag detection.

## Features
- ML model to classify job posts
- Red flag keyword detection
- Confidence score display
- Web interface using Flask
- Real vs Fake job prediction

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML
- CSS
- JavaScript

## How to Run

Install dependencies

pip install -r requirements.txt

Run application

python app.py

Open browser

http://127.0.0.1:5000
